In this course, I try to improve my Pyhton skills. First, I have started with Python Essentials (Variables, Type of Variables, etc).

## How could you clone this repository?

1. You should have GIT installed 
2. You try copying this commands on your CMD

``` bash
git clone https://github.com/Bidobelemti/python_for_data_science
cd python_for_data_science
```
3. You should see 3 folders
``` text
python_for_data_science
├───Data with Python
├───python essentials
└───Python fundamentals
```
